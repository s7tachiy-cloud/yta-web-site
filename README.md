# YTA WEB

シンプルなHP制作サービス「YTA WEB」の公式サイト。

## 公開 URL

https://s7tachiy-cloud.github.io/yta-web-site/

## 技術構成

- 静的 HTML + CSS のみ（JS フレームワーク不使用）
- フォント: Noto Sans JP（Google Fonts CDN）
- レスポンシブ: モバイル対応済み（768px ブレークポイント）
- ホスティング: GitHub Pages

## ページ構成

```
├── index.html             トップ
├── service.html           サービス紹介
├── pricing.html           料金
├── flow.html              ご利用の流れ
├── samples.html           サンプル一覧
├── faq.html               よくあるご質問
├── tokushoho.html         特定商取引法に基づく表記（noindex）
├── samples/               業種別サンプル HP（各 4 ページ）
│   ├── koumuten/          工務店サンプル
│   ├── gyouseishoshi/     行政書士サンプル
│   └── seitai/            治療院サンプル
└── css/style.css
```
