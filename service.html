サービス紹介

YTA WEB がご提供するHP制作サービスの内容です。

サービス概要

小さな事業者さまのためのシンプルなHP制作サービスです。

YTA WEB は、HPをお持ちでない中小事業者さま向けに、AIを活用した効率的な制作プロセスで、シンプルで信頼感のあるHPをご提供するサービスです。一般的なHP制作会社の3分の1〜5分の1の価格で、必要十分な品質のHPを納品いたします。

「HPがあったほうがよいのは知っているけれど、何から始めればよいかわからない」「見積りが高すぎる」「打ち合わせが面倒」といった声に応える形で設計しました。

対象となる業種・対象外の業種

YTA WEB は「メンテ不要設計」を採用しているため、すべての業種にお勧めできるわけではありません。

当サービスは「修正があまり発生しないHP」を低価格で長くお持ちいただくことを前提としています。情報更新が頻繁に必要な業種では設計と業種特性が合いませんので、事前に適合性をお伝えしています。

対象となる業種

メンテ不要設計と相性が良い業種

  • 工務店・建築・リフォーム会社情報・施工分野・連絡先が中心。年単位で内容が大きく変わりません。
  • 治療院・整体・サロン施術メニュー・院長紹介・予約導線が中心。基本構成は安定しています。
  • 士業（行政書士・税理士・社労士 等）業務範囲・代表者経歴・報酬目安が中心。長期的に内容が固定しやすい業種です。
  • クリーニング・配送・地域サービス料金表・対応エリア・営業時間が中心。情報更新の頻度が低い業種です。
  • その他、サービス内容が年単位で大きく変わらない事業個別にご相談ください。適合性を事前にお伝えします。

対象外の業種

頻繁な情報更新が必要な業種

  • 飲食店メニュー・季節商品・キャンペーンの変更が頻発するため、メンテ不要設計と相性が合いません。
  • 小売店商品入れ替えや在庫情報の更新が頻発するため、当サービスの設計には合いません。
  • 求人情報サイト掲載情報の入れ替えが本質的に必要な業種です。
  • ニュースサイト頻繁な記事更新が前提となるため、当サービスの対象外です。
  • EC サイト商品管理・決済・在庫機能が必要なため、専用サービスをお勧めします。

対象外の業種の方には他社サービスをご案内します。飲食店さま・EC をご希望の方には、ジンドゥー AI やペライチなど、ご自身で頻繁に更新できる SaaS 型サービスをお勧めしています。サービスのミスマッチで後悔されるより、最適な選択肢をお伝えするのが私たちのお約束です。

95%完成版送付モデル

完成品をご覧いただいてからご判断いただけます。

従来のHP制作では、お客様が打ち合わせや要件定義から始めて、長い時間をかけて完成形を作っていきます。しかしHPをお持ちでない方にとって、完成形をイメージするのは難しいものです。

YTA WEB では、Google Maps 等で公開されている事業情報をもとに、AI と業種別テンプレートを使って 95% 完成した状態のHP を先にお作りしてお送りします。残りの 5% は、電話番号・営業時間・固有のサービス内容などお客様にご確認いただく部分です。

「先に作ってお送りする」ことで、お客様の判断負荷をゼロにします。気に入っていただけたらご契約、合わなければそれまでです。一切のご請求はいたしません。

サービスに含まれるもの

  • HP一式の制作業種に合わせたデザインテンプレート、コピーライティング、スマートフォン対応、SEO 基本設定。
  • 独自ドメインの取得・設定お客様の屋号に合わせた独自ドメインを取得し、HPに紐付けます。
  • サーバー・ホスティングHPを公開し続けるためのサーバー費用。月額に含まれます。
  • HTTPS（SSL）対応セキュリティのための暗号化通信に対応します。
  • Google ビジネスプロフィール連携の助言地元検索で見つけてもらいやすくするための設定をお手伝いします。
  • 軽微な情報修正電話番号・営業時間・住所変更などの修正を月額料金内で対応します。

サービスに含まれないもの

明朗会計のため、含まれないものも明示します。

  • 大規模なリニューアル・追加ページ制作HPの構造を大きく変える作業や、新規ページの追加は別途お見積りとなります（5,000円〜）。
  • ロゴデザイン既存のロゴをご支給いただくか、簡易的な文字ロゴで対応いたします。本格的なロゴ制作は専門のデザイナーをお勧めします。
  • 商品撮影・取材掲載する写真はお客様にご用意いただきます。
  • SEO 対策の高度な施策基本的な設定は含みますが、競合の多いキーワードでの上位表示を保証するものではありません。
  • EC・予約システム等の高度な機能本サービスは「シンプルな紹介HP」に特化しています。複雑な機能が必要な場合はご相談ください。

修正対応の方針

「修正不要」を前提に、必要なときだけ対応します。

YTA WEB のHPは「修正があまり発生しない設計」を目指しています。事業者さまの基本情報（屋号・住所・電話番号・営業時間・サービスメニューの大枠）は、頻繁に変わるものではないからです。

そのうえで、以下の方針で運用しています:

月額料金内で対応する修正

  • 軽微な情報修正電話番号・営業時間・住所・代表者名の変更など、テキスト1〜2行程度の修正。

都度有料で対応する修正

  • 5,000円〜新規ページの追加、デザイン変更、画像差し替え、メニュー大幅改訂など。お見積りの上で承ります。

「追加料金がかかるかどうか」は、必ず作業前にお伝えします。事前にご納得いただいた金額のみ、お支払いいただく仕組みです。
