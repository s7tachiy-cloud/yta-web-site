お客様からいただくご質問にお答えします。

料金について

なぜこんなに安いのですか？

AIを活用した制作プロセスで工数を極限まで圧縮しているためです。一般的な制作会社は1社のHP制作に数十時間かけますが、YTA WEB では業種別テンプレートと AI 生成を組み合わせて大幅に時間を短縮しています。品質を犠牲にせず、価格を下げることに集中しました。

月額3,980円で何をしてくれますか？

サーバー・ドメインの維持、軽微な情報修正（電話番号・営業時間・住所変更など）、Google ビジネスプロフィール連携の助言などの保守運用が含まれます。HPを「公開し続ける」ためのコストとお考えください。

追加料金はかかりますか？

基本料金は制作費 150,000円 ＋ 月額 3,980円の2つで、これが契約期間中の固定費です。修正対応（新規ページ追加・デザイン変更・画像差し替え等）が必要な場合は別途 5,000円〜の明朗会計で承ります。ただし当社のHPは「修正不要のシンプル設計」を前提にしているため、ほとんどの方で追加費用は発生していません。修正の可能性がある場合は必ず作業前にお見積りをお伝えするので、事後の予期せぬ請求はありません。

なぜ「メンテ不要設計」と言えるのですか？

当社は、HPに掲載する情報を「変動頻度の低いもの」に絞った設計を採用しています。代表者・住所・電話番号・基本サービス内容といった「年単位で変わらない情報」が中心です。商品やメニューが頻繁に変わる業種（飲食店等）は対象外としているため、ほとんどのお客様は契約後に修正のご要望が発生していません。

修正が必要になった場合は？

都度有料（5,000円〜）の明朗会計で対応します。お見積りをお伝えしてから作業を開始するため、想定外の費用は発生しません。ただし、メンテ不要設計のため、ほとんどの方は契約後に修正のご相談をいただいていません。

初年度の総額はいくらですか？

12 か月利用された場合は、制作費 150,000円 ＋ 月額 3,980円 × 12 か月 = 197,760円（税込）です。2 年目以降は月額のみ（年間 47,760円）となります。

解約・契約について

本当に縛りなしで解約できますか？

はい。最低継続期間は設けておりません。ご解約のご連絡から月末まででサービスを停止します。違約金や解約手数料も一切ありません。

解約時のドメイン買取 30,000円とは何ですか？

YTA WEB ではお客様専用の独自ドメインを取得して運用しています。解約時には「ドメインを放棄」（無料）または「ドメインをお買い取りいただく」（30,000円）のどちらかをお選びいただきます。引き続きそのドメインで別のHPを運用したい場合に「買取」を選んでいただく形です。解約時の選択肢として事前にお伝えしているため、想定外の請求にはなりません。

完成版を見てから契約をやめることもできますか？

はい、可能です。仮 URL で完成版をご確認いただいた段階で、気に入らなければキャンセルしていただけます。その場合、一切のご請求は発生しません。

修正・更新について

修正は本当に必要ないのですか？

YTA WEB のHPは「修正があまり発生しない設計」を前提にしています。事業者さまの基本情報（屋号・住所・電話番号・営業時間・サービスメニューの大枠）は頻繁に変わるものではないからです。電話番号変更や営業時間変更などの軽微な修正は月額に含まれます。大幅な変更が必要な場合は別途有料（5,000円〜）で対応しますが、必ず事前にお見積りをお伝えします。

大きな変更を頼みたい場合はどうすればいいですか？

新規ページの追加、デザイン変更、画像の差し替えなどは都度有料（5,000円〜）で承ります。お見積りをお伝えしてから作業を開始するため、お客様にご納得いただいた上で進められます。

写真は自分で用意する必要がありますか？

はい、HPに掲載する写真はお客様にご用意いただきます。スマートフォンで撮影されたもので問題ありません。写真がない場合は、業種に合わせたフリー素材を活用したシンプルなデザインでご提案いたします。

制作・運用について

ドメインはなぜそちらが持つのですか？

運用上の管理（更新・SSL対応・DNS 設定）を一元化し、お客様の負担をゼロにするためです。お客様にドメイン管理の知識は不要です。なお、解約時には「放棄」または「30,000円でお買い取り」のいずれかをお選びいただけます。

納品までの期間はどのくらいですか？

最短 1〜2 週間です。ヒアリング後、3〜5 営業日で 95% 完成版をお送りし、ご確認・修正・契約・本番公開という流れで進みます。詳しくはご利用の流れをご覧ください。

EC サイトやネット予約システムは作れますか？

YTA WEB は「シンプルな紹介HP」に特化しており、EC や予約システムなどの高度な機能は含まれません。これらが必要な場合は、別の専門サービスをお勧めしています。

対応エリアはどこですか？

日本全国対応可能です。ヒアリングはオンライン（Zoom 等）で行うため、地域を問いません。
